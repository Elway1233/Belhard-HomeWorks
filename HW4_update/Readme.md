Задача: Telegram-bot для цветочного мазанина

Инструкция по запуску:
1. Создать проект
2. Скопировать все файлы в свою модель
3. В терминале запустить обучение python train.py --model deep
4. В коде вместо слов Your token,вставить токен от вашего бота
5. Запустить сам проект командой в терминале python telegram_bot.py

Так же имеется возможность посмотреть на уверенность модели в ответах,для этого можно запустить python debug.py
