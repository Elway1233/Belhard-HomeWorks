Задача: Скрещивать собаку и кота и выдавать гибрида

  Для реализации используется AE-GAN модель с Encoder, Generator, Discriminator описанные в файле model.py.
  
  В файле mix.py описан код морфинга(скрещивания).
  
  В файле train.py описан код обучения нашей модели.
  
Инструкция по запуску:

    1.Сначала запустить обучение нашей модели командой "python train.py"
    
    2.После обучения,создать директорию test_images и туда закинуть фото нашего кота и собаки с название cat.jpg,dog.jpg 
    и запустить "python mix.py" после этого в нашей директории создастся файл catdog_morphing.png
