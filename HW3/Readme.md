Задача: Система мониторинга сна на основе YOLOv5


## Инструкция по запуску:
1. В готовый проект yolov5 нужно добавить файл sleep_tracker.py по пути Z(ваше название диска):\YoloProject(ваше назваение проекта)\yolov5\
2. Скачать папку exp4 и так же добавить к своему проекту по пути Z(ваше название диска):\YoloProject(ваше назваение проекта)\yolov5\runs\train\
3. Создать папку в проекте под названием sleep_proofs по пути Z(ваше название диска):\YoloProject(ваше назваение проекта)\yolov5\
4. В файле sleep_tracker.py там где VIDEO_SOURCE расскоментировать строчку #VIDEO_SOURCE = 0 (если вы используете веб-камеру),если как я с телефона,то необходимой скачать приложение с плеймаркета IP Webcam
https://play.google.com/store/apps/details/?id=com.pas.webcam&hl=ru и оставить строку VIDEO_SOURCE = 'http://192.168.0.109:8080/video' и заменить мой ip address на свой(который будет указан в приложении)
5. Чтобы запустить проект,сначала нужно убедиться в терминале ,что вы находитесь в нужной директории,в моем случае Z:\YoloProject\yolov5> (для этого нужно воспользоваться cd yolov5) и запустить наш файл следующей командой python sleep_tracker.py

Результат:
У вас запустится окошко,где в режиме реального времени,модель будет опредлять спит человек или нет. Как только человек спит более 10 секунд,модель фиксирует время начала сна и делает скриншот,потом записывает это в файл и сохраняет скриншот в папку Z:\YoloProject\yolov5\sleep_proofs (у вас будет отличаться диск и название проекта Z:\YoloProject\)
